Description | System.Text.Json source generation provides compile-time generation of serialization code to enable high-performance, AOT-compatible JSON operations. It eliminates runtime reflection by generating typed serialization logic at compile time, making it essential for Native AOT deployments and high-performance applications. Source generation supports type-safe operations with unconstrained generics and automatic property naming policies, reducing the need for manual JsonPropertyName attributes.
